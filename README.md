# parcel-transformer-eta

[Parcel](https://github.com/parcel-bundler/parcel) (`2.x`) plugin to transform HTML templates with [eta](https://github.com/etajs/eta).

## Development

```bash
# Build the plugin
$ pnpm install
$ pnpm build

# Run example
$ cd example
$ node bundle.mjs
```
